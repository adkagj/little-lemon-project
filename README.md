# Little Lemon Restaurant Website

## Meta Front-End Developer Capstone Project

This project is the culmination of the Meta Front-End Developer Professional Certificate program. It showcases a fully functional restaurant website for Little Lemon, featuring modern design, responsive layouts, and interactive components.

- Form validation
- API integration

- HTML5
